# 调查结果与研究

## 要求和背景
- **项目目标：** [打造一个独立的前端应用项目]
    - 关键点：非个人网站，必须是一个有功能的产品。
    - 目的：通过项目的复杂度和视觉冲击力打动 HR/面试官。
- **目标受众：** [面试官 (主要)、用户]


## 设计研究
- **审美方向：** [建议：未来的生产力具象化 (Futuristic Productivity)]
    - **推荐方案**：**Lumina Flow - 沉浸式空间思维导图**。
    - **视觉风格**：深邃虚空背景 + 荧光/磨砂质感节点。摒弃传统白底黑字，采用类似 *Linear* 或 *Arc* 的现代极客美学，但更具艺术感。
    - **差异化**：将枯燥的“Todo List”变成“星系连接”。每一个任务是一个发光的星球，任务之间的关联是流动的光束。
