# 🌌 Lumina Flow - 空间思维引擎

[English](./README.md) | **简体中文**

**Lumina Flow** 是下一代空间思维导图工具，专为捕捉灵感与整理思绪而生。我们专注于极致的美学设计、流畅的交互体验与深度的专注力。

![Lumina Flow Preview](/preview.png)

## ✨ 核心功能

- **💎 磨砂玻璃 UI**
  采用现代化的半透明磨砂质感与环境光效 (Starlight/Nebula 主题)，带来令人惊艳的视觉体验。

- **🌌 无限空间画布**
  基于 React Flow 打造的无限画布，优化了丝滑的拖拽与缩放交互。

- **🔦 专注聚光灯**
  点击任意节点，自动将周围无关内容变暗模糊，仅高亮当前节点及其关联想法，助您深度思考。

- **🎋 智能引力布局**
  无论想法多杂乱，点击“智能整理”按钮，瞬间像引力般自动归位，呈现清晰的层级结构。

- **🌓 自适应主题**
  完美适配深色/浅色模式，支持跟随系统自动切换，每一处对比度都经过精心调校。

- **🌍 中英双语**
  原生支持简体中文与英文界面，所有提示与控件均已完全汉化。

## 🛠️ 技术栈

- **框架**: React 18 + Vite
- **核心库**: React Flow (@xyflow/react)
- **样式**: TailwindCSS + Custom Design System
- **状态管理**: Zustand
- **算法**: Dagre (图布局)
- **图标**: Lucide React

## 🚀 快速开始

1.  **克隆仓库**
    ```bash
    git clone https://github.com/0717lee/lumina-flow.git
    cd lumina-flow
    ```

2.  **安装依赖**
    ```bash
    npm install
    ```
    > **注意**: 需要 Node.js v18 或更高版本。

3.  **启动开发服务器**
    ```bash
    npm run dev
    ```

## 🎮 操作指南

- **创建节点**: 双击画布空白处。
- **连接想法**: 拖拽节点边缘的手柄进行连线。
- **专注模式**: 单击节点进入聚光灯模式，单击空白处退出。
- **智能整理**: 点击左下角工具栏的分支图标。
- **编辑内容**: 点击节点内部即可输入文本。
