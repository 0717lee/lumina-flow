# Lumina Flow

[English](./README.md) | **简体中文**

**Lumina Flow** 是一个空间化思维导图应用，适合记录灵感、拆分主题、整理复杂关系。它把无限画布、键盘优先的高效交互、本地持久化和轻量导出能力结合在一起，既能用来头脑风暴，也能做轻量规划。

![Lumina Flow Preview](/preview-zh-CN.png)

## 核心功能

- **无限空间画布**
  基于 React Flow 构建，可自由拖拽、缩放、连线，在大画布里组织复杂想法。

- **专注聚焦模式**
  选中节点后，会自动弱化无关内容，让当前想法和直接关联关系更容易阅读。

- **智能结构整理**
  支持一键自动布局，也支持用快捷键快速创建子节点和兄弟节点。

- **多画布工作区**
  可在同一个本地工作区里管理多个画布，支持重命名、切换，并自动保存在浏览器中。

- **命令式搜索与快捷键帮助**
  可用 `/` 或 `Ctrl/Cmd + K` 直接聚焦搜索，在标题、备注、标签中查找节点，用键盘切换结果，并通过 `?` 打开内置快捷键面板。

- **更丰富的节点信息**
  每个节点除了标题，还支持备注、标签、颜色和状态，既适合发散思考，也适合轻量任务规划。

- **反馈与撤销**
  导入和删除会显示 toast 提示，最近删除的节点或画布可立即撤销恢复。

- **可移植的导出能力**
  支持导出当前视图 PNG、整张画布 PNG，也支持按 JSON 导入导出单个画布或整个工作区。

- **主题、多语言与响应式面板**
  支持浅色、深色、跟随系统三种主题，以及英文和简体中文界面；在小屏设备上，工作区和属性面板会切换为底部抽屉。

## 快捷键

- `Tab`：基于当前节点创建子节点
- `Enter`：创建兄弟节点
- `Backspace` / `Delete`：删除当前选中节点
- `Ctrl/Cmd + Z`：撤销
- `Shift + Ctrl/Cmd + Z`：重做
- `/`：聚焦搜索
- `Ctrl/Cmd + K`：聚焦搜索
- `?`：打开快捷键帮助面板
- `Arrow Up` / `Arrow Down`：切换搜索结果
- 在搜索框中按 `Enter`：跳转到当前结果
- `Escape`：关闭搜索或帮助面板

## 技术栈

- **框架**: React 19 + Vite
- **核心画布**: React Flow (`@xyflow/react`)
- **状态管理**: Zustand
- **样式系统**: Tailwind CSS + 自定义设计 Token
- **图布局算法**: Dagre
- **导出能力**: `html-to-image`
- **图标**: Lucide React

## 快速开始

1. **克隆仓库**
   ```bash
   git clone https://github.com/0717lee/lumina-flow.git
   cd lumina-flow
   ```

2. **安装依赖**
   ```bash
   npm install
   ```
   说明：推荐使用 Node.js `18+`。

3. **启动开发服务器**
   ```bash
   npm run dev
   ```

4. **构建生产版本**
   ```bash
   npm run build
   ```

## 使用方式

- 双击画布空白处创建节点。
- 空画布会显示轻量引导，方便第一次进入时快速上手。
- 从节点手柄拖出连线，建立关系。
- 双击节点标题可直接编辑。
- 通过属性面板编辑备注、标签、颜色和状态。
- 通过搜索在标题、备注、标签中查找节点，并用键盘切换结果。
- 通过工作区面板新建画布，或导入导出 JSON。
- 删除节点或画布后会显示带撤销操作的 toast 提示。
- 通过底部工具栏切换主题、撤销重做，以及导出 PNG。
- 在小屏设备上，工作区和属性面板会以底部抽屉方式打开，避免遮挡主要画布。
